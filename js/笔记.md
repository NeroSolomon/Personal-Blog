## static、public、private

### static
只能类和子类本身直接调用，例如MyClass.staticFunc，可继承

### public
可继承、可以被类的实例、方法调用，不能被类本身调用

### private
原理：
```js
function MyClass {
    // 私有变量
    var p = 'content'
    this.getP = () => {
        return p
    }
}
```
不可继承，不能被类本身调用，只可以被类的方法调用


### replace正则替换
```js
var regx = /^-\s(')(\S+)(')$/
var a = "- '{{sys.code}}-redis'"
a.replace(regx, '- $2')
// 结果：'- {{sys.code}}-redis'
```
