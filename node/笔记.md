## node笔记

## babel-node
1. 如果你需要在node环境中使用es6的语法（import），而非common.js（require），就需要使用babel-node来执行命令
2. babel-node是集成在babel-cli中的，只需要安装babel-cli就好
3. babel-node如果没有全局安装的话需要用yarn执行，例如下面：yarn build
```json
{
  "scripts": {
    "build": "babel-node tools/build.js"
  }
}
```
4. 还需要新建一个.babelrc文件
```json
{
  "presets": [
    "es2015",
    "react"
  ]
}
```
