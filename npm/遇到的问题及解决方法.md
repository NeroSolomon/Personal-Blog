## 遇到的问题及解决方法
1.bad option: --test<br>
现象：使用--test会报错，但--prod或者--Fortest不会<br>
解决方案: 使用--isTest替换--test

2.如何同时运行多条命令
```
yarn add npm-run-all -D
```

```
npm-run-all --parallel [命令] [命令]
```
