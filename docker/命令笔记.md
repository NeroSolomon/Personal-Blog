## 展示image
```bash
docker images
```

or

```bash
docker image ls
```

## 将镜像下载到本地
```bash
docker save [image_id] > /example/test.tar 
```

## 将本地镜像文件导入
```bash
docker load < /example/test.tar
```

导入并改名
```bash
docker import /example/test.tar [image_name]:[image_tag]
```

## 重命名镜像tag
```bash
docker tag [image_id] [image_name]:[image_tag]
```
